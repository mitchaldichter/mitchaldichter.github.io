Heaviside function | Topic Playlist

The Heaviside function is a step function jumping from height zero to height one at the origin. It's useful in modelling circuits when there's an instantaneous change, like flipping a switch from open to closed in a circuit.

Definition and Uses for Piecewise Defined $f(t)$ | Explanation Video

A piecewise defined function $f(t)$ can be written using the Heaviside function $H(t)$ to make the calculation of the Laplace Transform easier.

\begin{equation} f(t) = \begin{cases} e^{t} & 0 < t \leq 2 \\ t-1 & 2 < t \leq 3 \\ \sin(t) & 3 < t \end{cases} \end{equation}

The Heaviside function is a step function.

\begin{equation} H(t) = \begin{cases} 0 & t < 0 \\ 1 & 0 \leq t \end{cases} \end{equation}

The Heaviside function can be shifted to jump at $t=c$ by changing the Heaviside function input to $H(t-c)$.

\begin{equation} H(t-c) = \begin{cases} 0 & t-c < 0 \\ 1 & 0 \leq t-c \end{cases} = \begin{cases} 0 & t < c \\ 1 & c \leq t \end{cases} \end{equation}

The Heaviside function can be flipped so it's 1 on the left and 0 on the right with $1-H(t-c)$.

\begin{equation} 1-H(t-c) = \begin{cases} 1-0 & t < c \\ 1-1 & c \leq t \end{cases} = \begin{cases} 1 & t < c \\ 0 & c \leq t \end{cases} \end{equation}

Two Heaviside functions can be subtracted from each other to be 1 in a range $c < t < d$ with $H(t-c) - H(t-d)$.

\begin{equation} H(t-c) - H(t-d) = \begin{cases} 0 & t < c \\ 1 & c \leq t < d \\ 0 & d \leq t \end{cases} \end{equation}

By using Heaviside functions, the piecewise defined function $f(t)$ can be written in a form that's easier to apply the Laplace Transform.

\begin{align} f(t) &= \begin{cases} e^{t} & 0 < t \leq 2 \\ t-1 & 2 < t \leq 3 \\ \sin(t) & 3 < t \end{cases} \\ &= e^{t}\big( 1-H(t-2) \big) \\ &\qquad+ (t-1)\big( H(t-3)-H(t-2) \big) \\ &\qquad+ \sin(t)H(t-3) \end{align}

Example 1 | Solution Video

Calculate the Laplace Transform for $H(t)$.

\begin{equation} \mathcal{L}\{ H(t) \} = \int_{0}^{\infty} H(t) e^{-st} \text{d}t \end{equation}

Example 2 | Solution Video

Calculate the Laplace Transform for $H(t-c)f(t)$.

\begin{equation} \mathcal{L}\{ H(t-c) \} = \int_{0}^{\infty} H(t-c) e^{-st} \text{d}t \end{equation}

Laplace Transform $\mathcal{L}\{ H(t-c) f(t) \}$ | Explanation Video

Use a substitution $\tau = t-c$ to change the bounds back to the normal 0 to $\infty$.

\begin{equation} \mathcal{L}\{ H(t-c) f(t) \} = \int_{0}^{\infty} H(t-c) f(t) e^{-st} \text{d}t = \int_{c}^{\infty} f(t) e^{-st} \text{d}t \end{equation}
