Repeated Eigenvalues | Topic Playlist

The repeated real nonzero eigenvalues case is one of several solution types for a two dimensional system of linear, constant coefficient, homogeneous differential equations. The types of solutions are called stable stars, unstable stars, stable degenerate nodes, and unstable degenerate nodes.

Solution | Explanation Video

When the system,

\begin{equation} \vec{x}' = A\vec{x} \qquad \lambda_{1} = \lambda_{2} \neq 0 \end{equation}

has repeated eigenvalues, there are two possible solutions.

The first solution type is called a star and the two-dimensional system will have the form,

\begin{equation} \vec{x}' = \begin{bmatrix} \lambda & 0 \\ 0 & \lambda \end{bmatrix}\vec{x} \end{equation}

In this case, every nonzero vector is an eigenvector since this is the identity matrix times the only eigenvalue $\lambda$.

A nice way to write the solution for a star is, \begin{equation} \vec{x}(t) = c_{1}e^{\lambda t}\begin{bmatrix} 1 \\ 0 \end{bmatrix} + c_{2}e^{\lambda t}\begin{bmatrix} 0 \\ 1 \end{bmatrix} = e^{\lambda t}\begin{bmatrix} c_{1} \\ c_{2} \end{bmatrix} \end{equation}

The second solution type is called a degenerate node and occurs when there is only one eigenvector. An example two-dimensional system has a form similar to a shear transformation.

\begin{equation} \vec{x}' = \begin{bmatrix} \lambda & m \\ 0 & \lambda \end{bmatrix}\vec{x} \quad \text{ where } m \neq 0 \end{equation}

This particular example has solution,

\begin{equation} \vec{x}(t) = c_{1}e^{\lambda t}\begin{bmatrix} 1 \\ 0 \end{bmatrix} + c_{2}e^{\lambda t}\begin{bmatrix} mt \\ 1 \end{bmatrix} \end{equation}

However, the solution in general requires trying a solution of the form,

\begin{equation} \vec{x}(t) = e^{\lambda t}(\vec{p}t + \vec{q}) \end{equation}

and solving for the unknown $\vec{p}$ and $\vec{q}$ vectors.

Phase Portraits | Explanation Video

For a star, $0 < \lambda$ results in an unstable star.

For a stable star, $\lambda < 0$.

All trajectories are red because every direction is an eigendirection.

For a degenerate node, $0 < \lambda$ results in an unstable degenerate node.

For a stable degenerate node, $\lambda < 0$.

There is only one red trajectory for the degenerate nodes because there is only one eigendirection.

Degenerate nodes with a diagonal eigendirection look the same except rotated.

Example 1 | Solution Video

Solve the following IVP, then draw a phase portrait with the eigendirections and the solution trajectory forward and backward in time.

\begin{equation*} \begin{bmatrix} x' \\ y' \end{bmatrix} = \begin{bmatrix} -3 & 0 \\ 0 & -3 \end{bmatrix}\begin{bmatrix} x \\ y \end{bmatrix} \quad \begin{bmatrix} x(0) \\ y(0) \end{bmatrix} = \begin{bmatrix} 2 \\ 7 \end{bmatrix} \end{equation*}
