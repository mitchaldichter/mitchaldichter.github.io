Verifying Solutions | Topic Playlist

To verify a function is a solution to an ODE, substitute the function into the ODE, calculate the derivatives, and see if the LHS and RHS are equal to each other.

Example 1 | Solution Video

Verify if the function solves the ODE or not.

\begin{equation} y' = 4y \qquad y(x) = 7e^{4x} \end{equation}

Example 2 | Solution Video

Verify if the function solves the ODE or not.

\begin{equation} x^{2}y' + y = 0 \qquad y(x) = x^{3} \end{equation}

Example 3 | Solution Video

Verify if the functions solves the ODE or not.

\begin{equation} y'' - 2y' + y = 0 \qquad y_{1}(x) = e^{x} \qquad y_{2}(x) = xe^{x} \end{equation}

Example 4 | Solution Video

Verify if the function solves the ODE or not.

\begin{equation} xy' + y = x \qquad y(x) = \frac{C}{x} + \frac{x}{2} \end{equation}

Example 5 | Solution Video

Verify if the function solves the ODE or not.

\begin{equation} y' + y = y^{2} \qquad y(x) = \frac{1}{Ce^{x}+1} \end{equation}
