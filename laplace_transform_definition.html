Laplace Transform | Topic Playlist

The Laplace Transform can be used to transform a linear initial value problem into another problem that can often be solved using only algebra and then applying the inverse Laplace Transform to find the solution to original initial value problem. The Laplace transform is useful in modelling circuits when an electrical pulse occurs or a switch is flipped.

Definition | Explanation Video

The Laplace Transform of a function $f(t)$, usually written as $F(s)$, is defined as

\begin{equation} \mathcal{L}\{ f(t) \} = F(s) = \int_{0}^{\infty} f(t) e^{-st} \text{d}t \end{equation}

There are some restrictions on the input function $f(t)$, such as not growing so fast that the integral evaluates to infinity, but most functions have a Laplace Transform.

\begin{equation} \mathcal{L}\{ 1 \} = \int_{0}^{\infty} e^{-st} \text{d}t \end{equation}

\begin{equation} \mathcal{L}\{ t^{n} \} = \int_{0}^{\infty} t^{n} e^{-st} \text{d}t \end{equation}

where $n$ is a positive integer.
